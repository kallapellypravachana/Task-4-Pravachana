# 🧠 General Knowledge Quiz (Python)
## 📌 Project Overview
This is a simple General Knowledge Quiz developed in Python. The program asks the user three questions, checks the answers, keeps track of the score, and displays the final result at the end.

This project demonstrates the use of Python fundamentals such as variables, user input, conditional statements, string methods, and formatted output.
## 🎯 Features
- Asks 3 General Knowledge questions.
- Accepts user input using `input()`.
- Ignores extra spaces using `.strip()`.
- Ignores uppercase/lowercase differences using `.lower()`.
- Calculates the user's score.
- Displays the final score using an f-string.
- Shows feedback based on the final score.
## 🛠️ Concepts Used
- Variables
- Input and Output
- `if`, `elif`, `else`
- Comparison Operators (`==`)
- String Methods (`.strip()` and `.lower()`)
- Score Counter
- f-Strings
## ▶️ How to Run
1. Install Python 3.
2. Save the program as `quiz.py`.
3. Open a terminal or command prompt.
4. Run the program:

```bash
python quiz.py
```
